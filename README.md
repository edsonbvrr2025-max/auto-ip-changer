# Auto_IP_changer V 2.2

Altere seu endereço IP automaticamente. Esta ferramenta é baseada no projeto Tor.

Como instalar esta ferramenta:

* Requisitos:

sudo apt-get install tor
pip3 install requests[socks]
Ou simplesmente execute autoTor, que instalará tudo.

1: git clone https://github.com/FDX100/Auto_Tor_IP_changer.git

2: cd Auto_IP_changer

3: python3 install.py

4: No terminal, digite (aut) em qualquer lugar que desejar.

5: Digite o número de vezes que deseja alterar o IP.

6: Digite quantas vezes deseja alterar seu IP.

* [0 a infinito]

6: Acesse seu navegador/computador e altere o proxy SOCKS para 127.0.0.1:9050.

7: Pronto!
